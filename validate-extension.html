🚀 Installation Instructions

Quick Setup (2 minutes)

  1. Open Chrome Extensions: Type chrome://extensions/ in your address bar
  2. Enable Developer Mode: Toggle the switch in the top-right corner
  3. Load Extension: Click "Load unpacked" and select this BookmarkX folder
  4. Verify Installation: Look for BookmarkX in your extensions list
  5. Test the Extension: Click the BookmarkX icon in your toolbar

🐛 Troubleshooting

Common Issues & Solutions

  • Extension won't load: Ensure all required files are present (check validation above)
  • Popup doesn't open: Right-click extension icon → "Inspect popup" → check console
  • Manager shows blank: Press F12 in manager tab → check console for errors
  • No bookmarks show: Ensure you have bookmarks saved in Chrome
