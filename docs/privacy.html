BookmarkX is a Chrome extension that helps users search and access their bookmarks quickly. We do not collect, sell, or share personal data.

Data We Access

  • Bookmarks API: Used to read and display your bookmarks so you can search and open them.
  • Storage: Used only to save lightweight preferences (e.g., UI settings, last search) locally on your device.
  • activeTab: Used only when you interact with the extension to open a bookmark in the current tab.

Data Collection & Transfer

BookmarkX does not collect, transmit, sell, or share personal data with third parties. All functionality operates locally within your browser.

Security

All code is packaged with the extension. We do not load or execute remote code.
